 Hostel Complaint Management Portal

Overview
The  Hostel Complaint Management Portal is a web-based application designed to streamline hostel complaint management for students and administrators. Students can submit complaints related to facilities such as maintenance, cleanliness, electricity, and water. Administrators can efficiently manage, track, and resolve complaints, improving transparency, accountability, and response time.

This project demonstrates  full-stack web development  using  Flask, SQLite, HTML, CSS, JavaScript, and Jinja templates.


 Features
  - Student Module:  
  - Submit complaints with detailed descriptions  
  - Track status of complaints  
  - View history of past complaints

 Admin Module:  
  - View all student complaints  
  - Assign, update, and resolve complaints  
  - Dashboard with complaint statistics  

 System Features:  
  - User authentication and session management  
  - Real-time status updates  
  - Clean and responsive UI using HTML, CSS, and JavaScript  
  - Dynamic content rendering using Jinja2 templates
